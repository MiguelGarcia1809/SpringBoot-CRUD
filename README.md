# SpringBoot-CRUD
Se realizó un CRUD de medicos de una clinica utilizando:
1. Front End
   - Bootstrap
   - JavaScript
2. Back End
   - Java Spring Boot
   - Hibernate
3. Base de Datos
   - MySQL
